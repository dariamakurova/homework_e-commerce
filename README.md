
# Домашнее задание модуля 3
## Описание:
 Проработка классов и объектов на основе темы e-commerce

## Установка:
Клонируйте репозиторий:
git clone https://github.com/dariamakurova/homework_e-commerce.git

## Использование

- cозданы классы:
  1) Создан абстрактный класс BaseProduct, миксин выводит в консоль название созданного экземпляра класса с параметрами
  2) от BaseProduct создан Product - товар с описанием
  3) Созданы 2 подкатегории - Смартфоны и Газонная трава
  4) Сздан абстрактный класс BaseCategory
  5) от BaseCategory созданы Category и Order
  
- реализована возможность создания объектов из json файла
- при добавлении товара в категорию осуществляется проверка объекта к классу Product или наследуемым

## Тестирование

Тестирование работы функций проводить через соответствующие модули тестирования в директории tests
Использовать фреймворк pytest
'poetry add --group dev pytest'
